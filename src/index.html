          Einstellungen

          Freesound.org Integration

          So erstellst du deinen API Key:

          1. Logge dich bei freesound.org ein.
          2. Gehe zu API Applications.
          3. Klicke auf "Apply for a new API credential".
          4. Gib der App einen Namen (z.B. "SonicFlow") und eine kurze Beschreibung.
          5. Nach der Erstellung kopiere den "Client secret/Api key".

          Download-Ordner

          Wähle aus, wo heruntergeladene Freesound-Dateien gespeichert werden sollen.
